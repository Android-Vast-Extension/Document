VastNetStateMgr

class VastNetStateMgr(context: Context) : BaseNetStateListener

VastNetStateMgr used to manage the view of different network status.

Currently mainly supports four view: CONTENT_STATE_SHOW_EMPTY_DATA, CONTENT_STATE_SHOW_NET_ERROR, CONTENT_STATE_SHOW_LOADING_ERROR, CONTENT_STATE_SHOW_LOADING.

If you want to customize the view, you can use setEmptyDataView, setNetErrorView, setLoadingView, setLoadingView, otherwise the default view will be used.
